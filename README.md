# Mark - 书签管理应用

一个简单、高效的书签管理应用，支持用户注册、登录、导入/导出浏览器书签，并保留原有的文件夹结构。

## 功能特性

- 用户注册与登录
- 从浏览器导入书签（支持 HTML 格式）
- 导出书签为 HTML 格式
- 保留书签文件夹结构
- 添加/编辑/删除书签
- 创建/重命名/删除文件夹
- 本地存储（当前版本）

## 如何使用

### 本地运行

1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 注册一个账户并登录

### 导入书签

1. 从 Edge、Chrome 或其他浏览器导出书签为 HTML 文件
2. 在应用中点击「导入书签」按钮
3. 选择导出的 HTML 文件
4. 书签将被导入并保留原有结构

### 导出书签

1. 点击「导出书签」按钮
2. 书签将自动下载为 HTML 文件
3. 该文件可以被导入到任何主流浏览器中
